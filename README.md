# Open Invoice Manager

Eine einfache und benutzerfreundliche Webanwendung zur Verwaltung von Kunden und zur Erstellung professioneller PDF-Rechnungen.

## Features

### Kundenverwaltung
- Kunden anlegen, bearbeiten und löschen
- Speicherung von Kundendaten (Name, Adresse, Kontaktinformationen)
- Übersicht aller Kunden
- Suchfunktion für schnellen Zugriff

### Rechnungserstellung
- Einfache Erstellung neuer Rechnungen
- Automatische Rechnungsnummerierung
- Auswahl von Kunden aus der Datenbank
- Hinzufügen mehrerer Rechnungspositionen
- Berechnung von Zwischensummen, Steuern und Gesamtbeträgen
- Angabe von Zahlungsbedingungen und Fälligkeitsdaten

### Produkt-/Dienstleistungsverwaltung
- Verwaltung von Artikeln und Dienstleistungen
- Hinterlegung von Preisen und Beschreibungen
- Schnelle Auswahl bei der Rechnungserstellung

### PDF-Generierung
- Automatische Erstellung professioneller PDF-Rechnungen
- Anpassbares Layout
- Einbindung von Firmenlogo und -daten
- Download und Versand der Rechnungen

### Rechnungsübersicht
- Übersicht aller erstellten Rechnungen
- Statusverwaltung (Offen, Bezahlt, Überfällig)
- Filterung und Sortierung
- Suchfunktion

### Einstellungen
- Hinterlegung von Firmendaten (Name, Adresse, Steuernummer, etc.)
- Anpassung von Rechnungsvorlagen
- Konfiguration von Zahlungsinformationen (Bankdaten)
- Mehrwertsteuer-Einstellungen
